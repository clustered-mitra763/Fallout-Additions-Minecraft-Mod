# Fallout Additions Minecraft Mod

> **Fallout Additions** — Immersive Minecraft mod that brings the post-apocalyptic world of Fallout into your game with new weapons, armor, creatures, locations, and mechanics.

[![Stars](https://img.shields.io/github/stars/pkmnono/Fallout-Additions-Minecraft-Mod)](https://github.com/pkmnono/Fallout-Additions-Minecraft-Mod) [![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)

**Fallout Additions Minecraft Mod** expands your Minecraft world with authentic Fallout content. Discover new weapons, power armor, radioactive zones, mutated creatures, Vaults, and many other iconic elements from the Fallout universe.

---
## 📦 What's Included
- Fallout Additions Mod
- New weapons and armor (including Power Armor)
- Mutated creatures and enemies
- Post-apocalyptic structures and Vaults
- New items, blocks, and mechanics
- Immersive Fallout atmosphere
- Compatible with latest Minecraft versions

## ❗️ Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Mod not loading            | Make sure Forge/Fabric is installed           |
| Game crashes               | Check compatibility with other mods           |
| Missing textures           | Install all required dependencies             |
| Antivirus warning          | Add the folder to exclusions                  |
| Performance issues         | Allocate more RAM in launcher settings        |
